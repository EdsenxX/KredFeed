### Instrucciones para iniciar el proyecto
1. Clonar el proyecto
```
  git clone https://github.com/EdsenxX/KredFeed.git
```
2. Dirigirse a la carpeta del proyecto
```
  cd KredFeed
```
3. Instalar las dependencias
```
npm install
```
4. Iniciar el proyecto
```
npm start
```
